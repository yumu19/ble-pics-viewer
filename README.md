# ble-pics-viewer

このプログラムは、交差点の信号制御機から発信される **高度化PICS (Pedestrian Information and Communication System)** のBLEアドバタイズを受信して可視化するツールです。  BLEスキャンにより受信した信号状態（赤／青／点滅青など）をリアルタイムで表示します。

## 高度化PICS（歩行者等支援情報通信システム）とは
- 歩行者支援のための情報通信システム
- iOS / Androidの「信GO！」アプリで、情報を取得できます
- BLEアドバタイズなので、スマートフォン以外のBLE端末でも受信できます
    - [信号の時刻表をつくる 2025 (1/2) :: デイリーポータルZ](https://dailyportalz.jp/kiji/shingo-jikokuhyo2025)
- 高度化PICS信号機の設置場所は、e-Govデータポータルの [高度化PICS整備交差点_csv](https://data.e-gov.go.jp/data/dataset/npa_20221124_0054/resource/6f7e83e1-be28-4030-961f-3b489c9f6ad8) にて公開されています

## 事前準備
- Pythonの動作環境を構築してください
  -  Python 3.14.0 にて動作確認
- 以下のパッケージをpipでインストール
```bash
pip install pyyaml bleak
```

## 起動
```bash
python ble-pics-viewer.py
```

## 法令遵守に関する注記（Legal Notice）

本プログラムが対象とする PICS の BLE アドバタイズ信号は、道路交通信号機などの公共インフラ設備から **不特定多数に向けて送信される公開電波** です。この電波には個人を特定可能な情報や秘匿通信は含まれず、以下の法令で保護される「通信の秘密」には該当しません。

- **電波法 第109条（通信の秘密）** に関して
  - 公衆に送信される無線通信の受信は制限されていません。  
- **電気通信事業法 第4条（通信の秘密）** に関して
  - 一方向のブロードキャスト通信（アドバタイズ送信）は「電気通信事業者の取扱中に係る通信」ではありません。

したがって、本プログラムによる **受信・可視化行為は合法的** であり、電波法・電気通信事業法・個人情報保護法のいずれにも抵触しません。

本プログラムは **受信専用（receive-only）** であり、信号を送信・改変・再送信・妨害する機能は一切含まれていません。
